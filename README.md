# Computerized Maintenance Management System Preventive
### Sistem ini dikembangkan untuk mempermudah akses dan pengelolaan laporan preventive maintenance dengan memastikan setiap mesin menjalani perawatan sesuai jadwal mingguan yang telah ditentukan.

## Folder :
### ServerApp : Back End
### InformasiMesin : Front End

### Halaman Detail Mesin pada aplikasi ini berfungsi untuk menampilkan informasi lengkap terkait mesin yang dipilih oleh pengguna. Informasi yang disajikan meliputi ID mesin sebagai identitas unik di dalam database, Nomor Entitas yang digunakan sebagai kode pengenal resmi, serta Nama Entitas yang menjelaskan nama atau deskripsi singkat dari mesin tersebut. Selain itu, halaman ini juga menampilkan Merek atau Tipe mesin yang berguna untuk keperluan teknis seperti perawatan dan penggantian suku cadang. Data mengenai jumlah kebutuhan pelumas atau bahan lain juga ditampilkan dalam satuan gram (Qty grm) untuk mendukung kegiatan preventive maintenance. Dengan adanya halaman ini, pengguna dapat memperoleh gambaran menyeluruh tentang kondisi dan spesifikasi setiap mesin secara cepat dan akurat, sehingga mempermudah dalam pengelolaan mesin di lingkungan produksi.

### Halaman **Kalender Preventive** menampilkan jadwal pemeliharaan mesin secara terstruktur dalam format kalender bulanan, mingguan, maupun harian. Melalui tampilan ini, pengguna dapat melihat semua agenda preventive maintenance yang telah direncanakan, termasuk informasi penting seperti nomor entitas mesin beserta keterangan aktivitasnya, misalnya "mesin rusak" atau "Pemeliharaan Rutin". Setiap kegiatan ditandai dengan warna hijau untuk membedakan status atau jenis kegiatan. Pengguna juga dapat mengakses tombol navigasi seperti “today” untuk kembali ke tanggal saat ini, serta tombol navigasi kiri dan kanan untuk berpindah antar bulan. Selain itu, tersedia tombol “Cetak Laporan Bulanan” yang memungkinkan supervisor mencetak rekap laporan preventive maintenance untuk keperluan dokumentasi atau evaluasi. Dengan adanya halaman ini, proses monitoring pemeliharaan mesin menjadi lebih mudah, terorganisir, dan terkontrol, sehingga meminimalisir risiko keterlambatan pemeliharaan dan kerusakan mesin yang tidak terdeteksi.
